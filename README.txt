Need to install:
	Backend: 
		- Java 8
		- Maven 3
		
		- MySql 5
		
		- Node 6
		- Npm 3
	
	Frontend:
		- Angular 2;
		

Before run app execute in command line:

	- cd path/to/your/main/folder
	- gulp mvn_clean_install
	- gulp npm_install_front

To execute the app in command line:
	
	- cd path/to/your/main/folder
	- gulp
